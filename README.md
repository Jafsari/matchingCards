# matchingCards
A 52 card deck matching game made with React JS.

Used the create-react-app generator.

### Description
In it's current state, the game allows for 2 people to play against each other in the matching game.
Users are able to view all of their matched cards as well as their current points.
The game terminates once there are no more cards left on the board.


### Instructions to run
In order to run app after cloning, you must run:
`npm install --save bootstrap`
